# Fitness Tracker Web App

This is a simple shareable iPhone-friendly fitness tracker web app.

## Run locally
1. Install Node.js
2. Open a terminal in this folder
3. Run:
   npm install
   npm run dev

## Deploy on Vercel
1. Create a free Vercel account
2. Import this folder as a new project
3. Deploy
4. Open the live URL in Safari on iPhone
5. Tap Share > Add to Home Screen

## Notes
- Each person keeps their own data in their own browser
- Export and Import lets you back up or move data
